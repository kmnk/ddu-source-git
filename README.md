# ddu-source-git
[ddu][ddu] source script for using git.

## Required
- [denops.vim][denops]
- [ddu.vim][ddu]
- [fugitive.vim][fugitive]
- [ddu-kind-file][ddu-kind-word]

## Configuration

```vim
call ddu#start({'sources': [{'name': 'git'}]})
```
